# Промпт для ежемесячного задания Claude Code (betting-calendar)

Создаётся на claude.ai/code/scheduled, запуск 1-го числа каждого месяца, репозиторий betting-calendar.

Календарь используется менеджерами рекламной платформы для прогноза пиков зрительского трафика на спортстриминговых сайтах (география — глобальная/смешанная).

---

В репозитории betting-calendar обнови файл events.json. Через веб-поиск найди крупные события на ближайшие 3 месяца от текущей даты, которые дают пики зрительского трафика на сайтах спортивных трансляций, по этим категориям:

- **Футбол** — АПЛ, Ла Лига, Серия А, Бундеслига, Лига 1, Лига чемпионов, Лига Европы, матчи сборных (Лига наций, отборы), финалы, суперкубки, топ-дерби (Эль Класико, манчестерское, миланское)
- **Теннис** — турниры Большого шлема, итоговые турниры ATP/WTA
- **UFC** — номерные турниры и крупные Fight Night
- **NFL** — старт сезона, плей-офф, Супербоул
- **Бокс** — крупные бои (титульные, хевивейт, звёздные реванши)
- **Киберспорт** — The International (Dota 2), CS2-мейджоры, чемпионат мира по LoL
- **Хоккей** — старты сезонов и плей-офф КХЛ и НХЛ, чемпионат мира
- **Баскетбол** — старт сезона и плей-офф НБА, Евролига (финал четырёх)
- **Формула-1** — этапы календаря (ключевые: Монако, Монца, ночные гонки, дебютные трассы, финал сезона)

Если в окно попадает Олимпиада (летняя или зимняя) — обязательно включи её отдельным событием с hot:3.

## Критерии hot (прогноз пика зрительского трафика, география глобальная)

- **hot:3** — максимальный одновременный пик зрителей мирового масштаба: финалы ЛЧ и ЧМ, Супербоул, старт АПЛ, Эль Класико, топ-бои PPV (UFC с звёздным мейн-ивентом, мегафайты в боксе), Олимпиада. Строго 5–8 штук на окно, не больше
- **hot:2** — заметный рост нагрузки: туры ЛЧ, матчи грандов в топ-лигах, номерные UFC без мегазвёзд, финальные уикенды ТБШ, ключевые этапы Ф-1, NFL Kickoff, открытие сезона НБА, Лига наций
- **hot:1** — умеренный или региональный рост: старты КХЛ/НХЛ (региональная аудитория), рядовые этапы Ф-1, теннис вне финалов, жеребьёвки, бои без топ-имён

Поправки на специфику стриминг-трафика:
- Киберспорт транслируется на бесплатных официальных Twitch/YouTube-каналах — на спортстриминговых сайтах пик слабый, обычно hot:1 (в note поясни)
- Многодневные турниры (ТБШ, ГП-уикенды): в note указывай, когда именно пик (финал, гонка воскресенья)
- Взвешивай по глобальному охвату: событие с огромной, но локальной аудиторией (КХЛ — RU/CIS, регулярка НХЛ/НБА — Северная Америка) не выше hot:2, обычно hot:1–2 с пометкой региона в note

Правила:
- Строго соблюдай текущую JSON-схему: `date` (ISO), `dateEnd` для диапазонов, `sport`, `title`, `note`, `hot` (1–3), `tbc`
- Названия sport ровно как выше: Футбол, Теннис, UFC, NFL, Бокс, Киберспорт, Хоккей, Баскетбол, Формула-1
- Заголовки и note на русском
- Дата официально не подтверждена или в источниках разночтения — обязательно "tbc":true и пометка в note
- Удаляй события, завершившиеся больше месяца назад
- Обнови lastUpdated на дату запуска
- Также замени встроенный блок DATA в index.html (между маркерами EVENTS DATA START/END) на те же данные, чтобы файл работал и без events.json
- Перед коммитом провалидируй JSON

Закоммить с сообщением "chore: update events for <месяц>" и запушь в main.

---

## Как добавить новый вид спорта

1. Дописать категорию в список выше и в перечень названий sport
2. Обновить промпт в задании на claude.ai/code/scheduled
3. Попросить Claude разово добавить события новой категории в events.json + index.html (или дождаться следующего автозапуска)

Фильтры на странице генерируются из данных автоматически — HTML при добавлении категорий править не нужно.
