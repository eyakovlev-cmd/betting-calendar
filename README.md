# Спортивный календарь — пики трафика

**Страница:** https://eyakovlev-cmd.github.io/betting-calendar/

Календарь крупных спортивных событий для менеджеров рекламной платформы. Показывает, когда на спортстриминговых сайтах ожидаются спайки зрительского трафика, — чтобы заранее предупредить клиентов и подготовить инфраструктуру к наплыву.

## Как пользоваться

В начале месяца просмотрите ближайшие события и отметьте важные у себя. В течение месяца заглядывайте за обновлениями: даты с пометкой «Дата уточняется» (tbc) подтверждаются позже, появляются новые события.

Уровни рейтинга:

| Уровень | Значение | Действие |
|---|---|---|
| **Топ** (hot:3) | Максимальный пик трафика мирового масштаба | Предупредить клиентов, проверить готовность инфраструктуры |
| 🔥 **Высокий** (hot:2) | Заметный рост нагрузки | Держать на радаре, учитывать в планировании кампаний |
| **Фоновый** (hot:1) | Умеренный или региональный рост | Обычно действий не требует |

География оценки — глобальная/смешанная: событие с большой, но локальной аудиторией (КХЛ, регулярка НХЛ) получает рейтинг ниже, чем сопоставимое глобальное.

## Данные

Источник данных — `events.json`:

```json
{"date":"2026-08-15","dateEnd":null,"sport":"UFC","title":"UFC 330","note":"...","hot":3,"tbc":true}
```

- `date` — дата начала (ISO), `dateEnd` — для диапазонов
- `sport` — категория (Футбол, Теннис, UFC, NFL, Бокс, Киберспорт, Хоккей, Баскетбол, Формула-1)
- `hot` — прогноз пика трафика 1–3
- `tbc` — дата официально не подтверждена

В `index.html` встроена копия данных (блок DATA между маркерами `EVENTS DATA START/END`) — страница работает и без `events.json`; лежащий рядом файл имеет приоритет.

## Обновление

Данные обновляются автоматически раз в месяц (1-го числа): поиск событий на ближайшие 3 месяца, правила и критерии — в [2026-07-22-update-task-prompt.md](2026-07-22-update-task-prompt.md). Обновляются оба файла: `events.json` и блок DATA в `index.html`.

Ручная правка: отредактируйте `events.json` и продублируйте изменения в блоке DATA в `index.html`, JSON должен остаться валидным.
